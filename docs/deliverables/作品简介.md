# AutoSolver Agent 作品简介

AutoSolver Agent 面向随机配送分配问题，采用“正式求解器 + 网页端 Agent”的双层架构。`solver.py` 负责官方评测热路径，在时间预算内输出合法分配方案；网页端 Agent 展示感知、规划、策略试跑、自动评估、回退和记忆沉淀过程。

系统核心特点是把组合优化求解过程组织为可观察的 Agent 闭环：根据样例画像选择策略组合，用 Critic 筛选候选方案，以 anytime 方式维护当前最优解，并通过 Self-Evolving Code Loop 展示策略生成、安全门、沙箱试跑和候选沉淀。

项目提供可复现的本地网页、测试命令、架构图、运行截图和官方提交记录。正式成绩只引用 `archive/runs/official_submit_20260520_132026_70222083.json`，页面中的本地评估信号仅用于解释 Agent 决策过程。
